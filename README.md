# TitsGUI (ImStyleUI) 🚀

Roblox için ImGui ve Iris kütüphanelerinden esinlenilerek geliştirilmiş, hem PC hem de Mobil destekli kompakt ve performanslı bir UI kütüphanesi.

## 📌 Özellikler
* 📱 **Mobil Desteği:** Dokunmatik ekranlar için optimize edilmiş element boyutları.
* 🎨 **Klasik ImGui Teması:** Geliştirici dostu koyu tema ve monospaced (kod) yazı tipi.
* 📦 **All-in-One:** Kütüphane bileşenleri tek bir script içinde hazır.
